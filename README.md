# 🛒 Bini-Dang Market

Application web de mini-commerce entre étudiants et résidents de Bini-Dang.

## Description

Bini-Dang Market est une plateforme web développée dans le cadre du cours
INF336 — Ingénierie des Applications Web, L3 Informatique,
Université de Ngaoundéré, 2025/2026.
Enseignants : Prof. DAYANG Paul et M. KOTVA Samuel.

Elle permet aux étudiants et résidents de vendre, acheter et échanger
des produits et services, avec messagerie, évaluations et paiement
via Orange Money et MTN Mobile Money.

Lien : https://github.com/falmataharoune80-boop/marché_bini_dang-1
